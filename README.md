Diseño Responsivo - Inicio de Sesión App Banco Pichincha
Descripción del Proyecto

Este proyecto consiste en el desarrollo del diseño responsivo de la pantalla de inicio de sesión de una aplicación móvil inspirada en la App del Banco Pichincha, utilizando Android Studio y XML.

El objetivo principal fue replicar la interfaz de usuario de una aplicación bancaria real aplicando buenas prácticas de diseño responsivo, organización visual y uso correcto de layouts en Android.

El diseño fue desarrollado únicamente a nivel de interfaz gráfica (UI), enfocándose en la estructura visual y experiencia de usuario.

Tecnologías Utilizadas
- Android Studio
- Java
- XML
- ConstraintLayout
- Material Design Components
- Gradle

Características del Diseño
- Diseño responsivo adaptable a diferentes tamaños de pantalla.
- Uso de ConstraintLayout para mejorar el rendimiento y la organización visual.
- Implementación de MaterialButton y CardView.
- Distribución proporcional de componentes mediante weights y constraints.
- Uso de iconos e ilustraciones similares a la interfaz original.
- Interfaz moderna inspirada en aplicaciones bancarias reales.

Componentes Implementados
Header Superior
- Logo del Banco
- Texto institucional
- Botón de ayuda

Sección Central
- Ilustración principal dentro de una CardView
- Botón flotante “Ver mi QR”

Accesos Rápidos
- Verificar transferencia
- Pagar con QR
- Clave Digital

Métodos de Inicio de Sesión
- Usuario y contraseña
- Huella / Face ID
- PIN de 6 dígitos

Diseño Responsivo

Para garantizar que la interfaz se adapte correctamente a diferentes dispositivos se utilizaron:
- ConstraintLayout
- layout_weight
- tamaños relativos
- márgenes consistentes
- constraints horizontales y verticales
- componentes Material Design
